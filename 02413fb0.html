welcome to snapsite

Encapsule/snapsite is an experimental command line tool for compiling React JS websites from configuration files that's being developed to reduce the time/effort required to build sophisticated documentation portals for technical projects.

pre-release: snapsite v0.0.9 is not ready for prime time. There are still a number of sigificant features still to be implemented (e.g. the generated Node.js server) and quite a bit of churn is expected here over the coming months.

Please follow @Encapsule on Twitter for news and updates.

what are you looking at?

What you're looking at now is a simple test site built with the Encapsule/snapsite v0.0.9 website generator tool for Node.js.

The tool runtime reads configuration and JSX modules from a local filesystem clone of your project repository, generates the site, and writes the output back to the local filesystem.

You can look at the input sources for this website in the project's ./doc directory.

And, you can inspect the current client-side application generated by this process by visiting the project's #gh-pages branch.

To learn more, please read the tool runtime overview that explains the set of tasks automated.
